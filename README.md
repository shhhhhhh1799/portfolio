# 🧮 AI 수학 튜터 시스템 (Mathtutor)

카메라로 수식을 인식하고 계산한 뒤, LLM(Gemma)을 활용해 풀이 설명을 생성하고 음성으로 출력하는 AI 수학 튜터 시스템입니다.  
수학 문제를 손으로 써서 보여주기만 하면 인식부터 음성 설명까지 한 번에 진행됩니다.

---

## 📌 프로젝트 개요

이 프로젝트는 수식 인식, 자동 계산, 자연어 풀이 설명, 음성 출력까지 통합한 **AI 기반 수학 학습 보조 시스템**입니다.  
사용자는 카메라 또는 음성으로 수학 문제를 입력할 수 있으며, 시스템은 단계별로 계산 후 음성으로 설명해줍니다.

> 교육, 튜터링, 자율 학습 등에서 활용할 수 있는 AI 교재 보조 기술을 목표로 합니다.

---

## ⚙️ 주요 기능

- 📸 **수식 이미지 인식** (Mathpix API 사용)
- 🧠 **수식 계산** (SymPy 사용 – 미분, 적분, 방정식, 푸리에 변환 등 지원)
- 💬 **자연어 풀이 생성** (Gemma 3B LLM 연동)
- 🔈 **음성 출력** (gTTS + Speech Rule Engine)
- 🎤 **음성 입력 지원** (Whisper를 통한 수식 음성 인식)

---

## 🧰 사용 기술

| 영역 | 기술 |
|------|------|
| 프로그래밍 | Python |
| 컴퓨터 비전 | OpenCV |
| 수식 OCR | Mathpix API |
| 수식 계산 | SymPy |
| 자연어 생성 | Gemma 3:12B via Ollama |
| 음성 처리 | Whisper (STT), gTTS (TTS), SRE (수식 발음 규칙) |
| 환경 | Raspberry Pi, Windows, Linux 가능 |

---

## 🚀 실행 흐름

1. 사용자 입력 (이미지 or 음성)
2. → Mathpix로 수식 인식
3. → SymPy로 수식 분석 및 계산
4. → Gemma LLM으로 풀이 설명 생성
5. → gTTS + SRE로 음성 출력

---

## 📁 주요 파일 설명

| 파일명 | 설명 |
|--------|------|
| `main.py` | 전체 시스템 통합 제어 |
| `camera_capture.py` | 카메라 이미지 캡처 |
| `ocr_mathpix.py` | 수식 이미지 → LaTeX 변환 |
| `solver.py` | SymPy로 수식 계산 |
| `explanation_llm.py` | Gemma로 풀이 문장 생성 |
| `speech_output.py` | 음성 출력 처리 (gTTS + SRE) |
| `whisper_input.py` | 음성 인식(STT) 처리 |
